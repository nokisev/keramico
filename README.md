## Зависимости

Gin <code>go get -u github.com/gin-gonic/gin</code>

MySQL Driver <code>go get -u github.com/go-sql-driver/mysql</code>

JWT <code>go get -u github.com/golang-jwt/jwt/v5</code>

## Документация

Пользовательскую документацию можно получить <a>по этой ссылке</a>

### Описание коммитов

| Название | Описание                                                        |
| -------- | --------------------------------------------------------------- |
| build    | Сборка проекта или изменения внешних зависимостей               |
| sec      | Безопасность, уязвимости                                        |
| ci       | Настройка CI и работа со скриптами                              |
| docs     | Обновление документации                                         |
| feat     | Добавление нового функционала                                   |
| fix      | Исправление ошибок                                              |
| perf     | Изменения направленные на улучшение производительности          |
| refactor | Правки кода без исправления ошибок или добавления новых функций |
| revert   | Откат на предыдущие коммиты                                     |
| test     | Добавление тестов                                               |
